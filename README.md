# Whatsapp Bot

### Feature
- Self Bot (True/False) Command `config.json`

### Command

- Ask Gemini AI
  `.gemini <question>`
- Add Member (Group)
  `.add <number>`
- Kick Member (Group)
  `.kick <@mention>`
- Promote Member to Admin (Group)
  `.promote <@mention>`
- Demote Admin to Member (Group)
  `.demote <@mention>`
- Close Chat Room (Group)
  `.chat-close`
- Open Chat Room (Group)
  `.chat-open`
- Enable Auto Delete Link (Group)
  `.antilink-true`
- Disable Auto Delete Link (Group)
  `.antilink-false`
